# brainfuck-go

## About
brainfuck-go 는 Go 로 작성 된 Brainfuck 언어의 인터프리터입니다. 걍해봄.

## Usage
### Compile
```
go build ./cli/bf.go
```
### Execute
```
./bf filename
```

## Example
```
./bf ./hello.bf                                                                                                                                                                                                                                                                       00:23:44
Hello, Brainfuck-Go
```
